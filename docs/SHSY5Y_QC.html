Aligned Data

Trimmed reads were aligned to the human genome assembly GRCh38.p13 from Ensembl Release 98 using STAR 2.7.0d and summarised to each gene using featureCounts. These read counts are read into a DGEList object for gene-level analysis.

dgeList <- here::here("data", "2_alignedData", "featureCounts", "genes.out") %>%
  read_delim(delim = "\t") %>%
  set_names(basename(names(.))) %>%
  as.data.frame() %>%
  column_to_rownames("Geneid") %>%
  as.matrix() %>% 
  set_colnames(str_remove(colnames(.), "Aligned.sortedByCoord.out.bam")) %>%
  DGEList(
    samples = tibble(sample = colnames(.)) %>%
      left_join(dplyr::filter(filename, read == "R1")),
    genes = grGenes[rownames(.)] %>%
      as.data.frame() %>%
      dplyr::select(
        chromosome = seqnames, start, end, 
        gene_id, gene_name, gene_biotype, description, 
        entrezid, gc_content, length
      )
  ) %>%
  .[!grepl("rRNA", .$genes$gene_biotype),] %>%
  calcNormFactors()
SH_cell <- grepl(paste(unique(SH_metadata$sample), collapse="|"),dgeList$samples$sample)  
SH_dgeList <- dgeList[,SH_cell]
minCPM <- 1.5
minSamples <- 3
SH_dgeList %<>% .[rowSums(cpm(.) >= minCPM) >= minSamples,]

At this step, genes were removed if:

  • They were not considered as detectable (CPM < 1.5 in > 33 samples). This translates to > 58 reads assigned a gene in all samples from one or more of the treatment groups
  • The gene_biotype was any type of rRNA.

These filtering steps returned gene-level counts for 13,941 genes, with total library sizes between 38,497,645 and 84,710,972 reads assigned to genes.

gcInfo <- SH_dgeList$counts%>%

A run length encoded (RLE) vector was formed for each sample taking the number of reads for each transcript as the run lengths, and both the GC content and length of each transcript as the values. Transcript lengths were transformed to the log10 scale due to the wide variety of lengths contained in the transcriptome.

From these RLEs, the mean GC and mean length was calculated for each sample, and these values were compared to the proportion of raw reads with > 70% GC, taking these values from the R1 libraries only.

Model Fitting

minLfc <- log2(2)
alpha <- 0.01
fit <- glmFit(SHContro_dgeList)
topTables_SHContr <- colnames(d)[2:6] %>%
  sapply(function(x){
    glmLRT(fit, coef = x) %>%
      topTags(n = Inf) %>%
      .[["table"]] %>%
      as_tibble() %>%
      arrange(PValue) %>%
      dplyr::select(
        gene_id, gene_name, logFC, logCPM, PValue, FDR, everything()  
      ) %>%
      mutate(
        coef = x,
        bonfP = p.adjust(PValue, "bonf"),
        DE = case_when(
          bonfP < alpha ~ TRUE,
          FDR < alpha & abs(logFC) > minLfc ~ TRUE
        ),
        DE = ifelse(is.na(DE), FALSE, DE)
      )
  }, simplify = FALSE)

Models were fit using the negative-binomial approaches of glmFit(). Top Tables were then obtained using likelihood-ratio tests in glmLRT(). These test the standard \(H_0\) that the true value of the estimated model coefficient is zero. These model coefficients effectively compares: 1. 0hr DMSO treated cells vs 0H DMEM treated cells 2. 15 hrs DMSO treated cells vs 0H DMEM treated cells 3. 30 hrs DMSO treated cells vs 0H DMEM treated cells 4. Difference between 0 and 15 hrs DMSO treated cells vs differences between 0 and 15 hrs DMEM treated cells 3. Difference between 0 and 30 hrs DMSO treated cells vs differences between 0 and 30 hrs DMEM treated cells

For enrichment testing, genes were initially considered to be DE using: a. A Bonferroni-adjusted p-value < 0.01 b. An FDR-adjusted p-value < 0.01 along with an estimated logFC outside of the range \(\pm \log_2(2)\).

Auophagy flux

As part of the study, the autophagic flux of cells were measured with an tandem-fluorescent LC3 assay (tf-LC3), which output a RG ratio for each sample and higher ratio implies higher autophagic activity.

In both control arms, autophagic activities of cells barely changed through the 30hr time course. mTOR-inhibited cells’ autophagic activity increased steadily from 1hr to 15hr but plateaued after that. Starved cells, on the other hand, experienced dramatic rise in autophagic flux for the entire duration.
