Trimmed reads were aligned to the human genome assembly GRCh38.p13 from Ensembl Release 98 using STAR 2.7.0d and summarised to each gene using featureCounts. These read counts are read into a DGEList object for gene-level analysis.

During this process, genes were removed if:

  • They were not considered as detectable (CPM < 1.5 in > 27 samples). This translates to > 93 reads assigned a gene in all samples from one or more of the treatment groups
  • The gene_biotype was any type of rRNA.

# gcDev is the standard deviation of observed GC frequencies from the expected GC frequencies, as obtained from the known transcriptome.
gcDev <- gc %>%
    left_join(filename) %>%
    group_by(sample, Filename) %>%
    mutate(Freq = Count / sum(Count)) %>%
    left_join(
        getGC(gcTheoretical, "Hsapiens", "Trans")) %>%
    dplyr::rename(actual = Hsapiens) %>%
    mutate(res = Freq - actual) %>%
    summarise(ss = sum(res^2), n = n()) %>%
    ungroup() %>%
    mutate(sd = sqrt(ss / (n - 1)))

It was identified in the previous steps that the percentage of libraries containing greater than 70% GC content is larger than the expected value in almost half of the libraries.

A run length encoded (RLE) vector was formed for each sample taking the number of reads for each gene as the run lengths, and both the GC content and length of each gene as the values. Gene lengths were transformed to the log10 scale due to the wide variety of lengths contained in the genome.

From these RLEs, the mean GC and mean length was calculated for each sample, and these values were compared to the proportion of raw reads with > 70% GC, taking these values from the R1 libraries only.

Models were fit using the negative-binomial approaches of glmFit(). Top Tables were then obtained using likelihood-ratio tests in glmLRT(). These test the standard \(H_0\) that the true value of the estimated model coefficient is zero. These model coefficients effectively compares: 1. 0hr DMSO treated cells vs 0H DMEM treated cells 2. 0hs DMSO treated cells vs 30hrs DMSO treated cells 3. Difference between 0 and 30 hrs DMSO treated cells vs differences between 0 and 30 hrs DMEM treated cells

Genes were considered to be DE using: a. A Bonferroni-adjusted p-value < 0.01 b. An FDR-adjusted p-value < 0.01 along with an estimated logFC outside of the range \(\pm \log_2(2)\).
